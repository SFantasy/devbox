DevBox
---

An Desktop application for developers.

![devbox](./screenshot/home.png)

## Building

```sh
$ git clone https://github.com/SFantasy/devbox
$ cd devbox
$ npm install
$ npm run webpack:build
$ npm start
```

## Development

```sh
$ npm run webpack:watch
// Open another tab in your terminal
$ npm start --debug
```
